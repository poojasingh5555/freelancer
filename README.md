Freelkancer is a full-stack MERN (MongoDB, Express, React, Node.js) freelancing platform that allows clients to post projects and freelancers to bid on them. 
The platform includes real-time chat, project management, and secure authentication to provide a smooth freelancing experience.
# Key Features
User Authentication – Signup/login with JWT and secure password hashing.
Project Posting & Bidding – Clients can post projects; freelancers can apply and bid.
Real-Time Chat – Built with Socket.IO for seamless communication between clients and freelancers.
Project Management – Track applications, submissions, and project status.
Responsive UI – Built with React and Material UI for an intuitive experience.
# Tech Stack
Frontend: React, BootStrap, Axios, React Router
Backend: Node.js, Express.js, JWT, bcrypt
Database: MongoDB with Mongoose
Real-Time: Socket.IO
Deployment: Render
